Controle de Despesas Pessoais

## Create a Venv

```bash
python -m venv venv
```

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requeriments.txt
```

## Running

```bash
py manage.py runserver
```
